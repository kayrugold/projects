LATTICE EXPLORER v0.0.3

SYSTEM: Modular Geometry Visualizer

The Mission: To bridge the gap between abstract arithmetic and visual intuition. The Lattice Explorer turns Number Theory into a map you can navigate.

Visualizing the Code

We usually think of factors as numbers in a list. But in the Forge, we view them as intersections in space. This engine renders the "DNA" of a number $N$ by plotting its geometric constraints on an infinite 2D plane.

The Layers:
> CORE: Hyperbolas (xy = N)
> DIAMONDS: Factor summation bounds
> PARABOLAS: Quadratic residue paths
> MODULUS: The underlying grid texture

Technical Specs

  • Touch Physics: Full pan and zoom support for mobile navigation (tested on the road).
  • Web Worker Threading: Offloads the heavy factorization and plotting logic to a background thread to keep the UI buttery smooth.
  • S-Candidate Plotting: A dedicated layer for visualizing potential 'S' values against the lattice structure.
