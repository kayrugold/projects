PURPOSE:
To determine if a massive number (like 10^1,000,000,000 + 61) is divisible by any smaller prime numbers, without calculating the massive number itself.
THE METHOD (STRIDE THEORY):
We treat the number line as an infinite tape. Every Prime Number is a "Walker" with a specific stride length (3, 5, 7, etc.).
EXAMPLE: Is 17 Prime?
1. Target Index: (17+1)/2 = 9.
2. The 3-Walker starts at 5. Steps: 5 -> 8 -> 11.
> Does it hit 9? NO.
3. The 5-Walker starts at 13.
> Does it hit 9? NO (Started too late).
RESULT: Since no Walker stepped on Index 9, the number 17 is Prime.
HOW IT GENERATES PRIMES:
It uses a "Segmented Sieve." It assumes all odd numbers are prime, then systematically marks off multiples (3, 9, 15...). Anything not marked is a Prime Walker.
THE INPUTS:
- Base/Exp/Add: Defines your Candidate Number (Base^Exp + Add).
- Walker Limit: How deep to scan. 10,000,000 checks all primes up to 10M.
THE RESULTS:
- SNIPED (Red): The number is Composite (divisible).
- SURVIVED (Green): The number has NO factors in this range. It is a strong candidate.
