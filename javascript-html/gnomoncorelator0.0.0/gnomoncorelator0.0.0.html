Core Formulas

  • Odd Number: N = 2k + 1
  • Sundaram Index: k = i + j + 2ij
  • Factors: A = 2i + 1, B = 2j + 1

Gnomon Geometry

Every composite odd number N = A × B is the difference of two squares:

  • Outer Square: ((A+B)/2)²
  • Inner Square: (|A-B|/2)²

Shortcuts

  • Mod 3 Wall: Any number divisible by 3 is marked Amber.
  • Index Badge: The tiny number in the corner of blue squares is 2n - 1, the Gnomon of that square.

Input Guide

Enter values in the i, j, or k boxes below the grid to reverse-lookup any Gnomon, even outside the visible table.
