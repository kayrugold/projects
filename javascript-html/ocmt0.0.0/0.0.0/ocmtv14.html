1. Finding a Number (N)

Type any number (e.g. 15375) or expression (e.g. 2^127-1) into the Find box.

  • Row (k): ceil((sqrt(N) - 1) / 2)
  • Col (j): (N/P2 - 1)/2
  • Gap (b): k - j (distance from square)

2. Layouts

  • SPINE: Squares are in Column 0.
  • DIAG: Standard multiplication table.
