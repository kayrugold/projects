Gnomon Indexing Manual ×

1. The Philosophy of Indexing

Factoring is not about calculation; it is about finding the unique address (k, b) on the infinite multiplication table.

Example: 1007 has the unique index 36.17.
This means it is found at Anchor 36, Gap 17.

36² - 17² = 1296 - 289 = 1007

2. The Gnomon Key (k, b)

The coordinates instantly reveal the factors P1 and P2 without needing to search the value N.

  • Row Index k: The Gnomon Key. It determines the larger factor P2 = 2k + 1.
  • Inner Gap b: This determines the smaller factor P1 = P2 - 2b.
