1. Finding Coordinates

To find where any number N lives, calculate its address:

Step 1: The Row (k)
The Row Index corresponds to the larger factor P₂ = 2k+1.
k = ceil((sqrt(N) - 1) / 2)

Step 2: The Column (j or b)
Depends on your layout mode:

  • If "Col" is Diagonal Index (j):
    Standard multiplication table index where P₁ = 2j+1.
    j = ((N / (2k+1)) - 1) / 2
  • If "Col" is Inner Gap (b):
    Spine Mode index (distance from square).
    b = k - j

2. Summary Workflow

  1. Divide N by the row's value (2k+1) to get the smaller factor (P₁).
  2. Convert factor to index: j = (P1 - 1) / 2.
  3. Subtract to find gap: b = k - j.

3. Layout Modes

  • SPINE (Left-Aligned): Squares are in Column 0. The X-axis is the Inner Gap (b).
  • DIAG (Standard): Standard multiplication table. Squares are on the diagonal (k=j).
