1. Cruise Control Scanning

Enter a number to perform a Method of Differences scan. The system calculates the starting square and drives down the number line adding integers until it fits the "Picture Frame."

2. Coordinates

  • Anchor (a): Outer Square size.
  • Gap (b): Inner Hole size.
