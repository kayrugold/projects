1. THE TAPE CONCEPT

Every composite number has a unique seat on an infinite tape. We call this the Pure ID (L).

2. UNPACKING AN ID

To find where ID 343 lives, the computer uses the Triangular Root formula to find the Row (k):

k = ceil( (sqrt(8L + 1) - 1) / 2 )

Example for ID 343: It lands on Row 26, which starts at ID 326. The Gap (b) is the distance from the start: 343 - 326 = 17.

3. THE GEOMETRY
  • Row (k): Determines the Large Factor (P2 = 2k + 1).
  • Gap (b): Distance from the Spine. Determines the Small Factor (P1 = P2 - 2b).
  • Anchor (a): The center of the Frame (a = P2 - b).
